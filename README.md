# 🧠 Blog Backend (MERN)

This is the backend of a Fullstack Blog App built using Node.js, Express, MongoDB, and Cloudinary.

---

## 🚀 Features
- Create posts
- Delete posts
- Store data in MongoDB
- Upload images using Cloudinary

---

## 🛠 Tech Stack
- Node.js
- Express
- MongoDB (Mongoose)
- Multer
- Cloudinary

---

## ⚙️ Setup

1. Install dependencies:
npm install

2. Create a `.env` file:
MONGO_URI=your_mongodb_url
CLOUD_NAME=your_cloudinary_name
API_KEY=your_api_key
API_SECRET=your_api_secret

3. Run server:
node server.js

---

## 🌐 API Routes

GET    /api/posts  
POST   /api/posts  
DELETE /api/posts/:id  

---

## 🚀 Deployment
Deployed on Render

---

## 🧠 Flow
React → Express → MongoDB → Cloudinary → React
